from appium import webdriver 导入远程服务

变量 = {} 创建一个字典

变量['deviceName'] = '127.0.0.1:62001' 设备名称

变量['platformName'] = 'Android' 平台名称

变量['platformVersion'] = '5.1.1' 平台版本（安卓版本）

变量['app'] = r'app地址' res'app所在地址+名称'

变量['appPackge'] = 'com.XX.XX' 使用As 打开XML查看

变量['appActivity'] = 'xx' app活动

driver = webdriver.Remote('http://localhost:4723/wd/hub',变量)

wd 可以理解是WebDriver 的缩写

hub 是指主（中心） 节点，在selenium 分布式里中心节点

单词：device设备 driver服务
